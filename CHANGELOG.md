# Changelog

All notable changes to `kanban` will be documented in this file.

## 1.0.0 - 2024-11-15

### Added
- Initial release
- Board management with CRUD operations
- Issue management with CRUD operations
- FilamentPHP 3 resources for Boards and Issues
- Configurable statuses and priorities
- User assignment functionality
- Tag support for issues
- Due date tracking
- Soft delete support
- Advanced filtering options
- Bulk actions
- Color-coded boards
- Priority badges
- Overdue issue detection
- Laravel 12 support
