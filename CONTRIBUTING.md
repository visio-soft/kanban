# Contributing to Kanban

Thank you for considering contributing to the Kanban package!

## Development Setup

1. Clone the repository
2. Install dependencies: `composer install`
3. Run tests: `composer test`
4. Format code: `composer format`

## Pull Request Process

1. Update the README.md with details of changes if applicable
2. Update the CHANGELOG.md following the existing format
3. Ensure all tests pass
4. Follow PSR-12 coding standards
5. Write clear commit messages

## Code Style

This project follows PSR-12 coding standards. Use Laravel Pint to format your code:

```bash
composer format
```

## Reporting Bugs

Please use the GitHub issue tracker to report bugs. Include:
- PHP version
- Laravel version
- FilamentPHP version
- Steps to reproduce
- Expected behavior
- Actual behavior

## Security Vulnerabilities

If you discover a security vulnerability, please email security@visiosoft.com instead of using the issue tracker.
